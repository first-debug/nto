Пояснительная записка

Для запуска использовать launch.bat

Для доступа ко всей информации нужно зайти под администратором:
	в левом верхнем углу Файл->Вход для сотрудников
	логин: admin1
	пароль: admin
Для доступа к экрану заявок для сотрудников служб:
	в левом верхнем углу Файл->Вход для сотрудников
	логин: employee1
	пароль: 1234Emp
ПРИМИЧАНИЕ1: теперь только администратор может добавлять какие-либо события
ПРИМИЧАНИЕ2: -1 в таблице помещений означает, что информация неизвестна
Главый экран для посетителей
Экран со всеми заявками - для сотрудников служб(доступ получить могут только они)
Экран с полной информацией обо всём - для администраторов

!На рабочем столе сотрудников служб визуальный баг при изменении фильтра
