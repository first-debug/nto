-----------------------------------Пояснительная записка----------------------------------
Весь новый функцианал тут -> в левом верхнем углу Файл -> Вход для сотрудников
	                         логин: admin1
	                         пароль: admin

Верхний левый углу Помощь -> О программе - пояснительная записка
Есть фильтр в окне с о всеми помещениями
Время в окне добавления кружков можно вводить в нескольких форматах (чч:мм/чч;мм/чч.мм)
!Если не получается запустить: папку 'Java' переместите в Program Files/Program Files(x86)
------------------------------------------------------------------------------------------------
Для доступа ко всей информации нужно зайти под администратором:
	в левом верхнем углу Файл->Вход для сотрудников
	логин: admin1
	пароль: admin
Для доступа к экрану заявок для сотрудников служб:
	в левом верхнем углу Файл->Вход для сотрудников
	логин: employee1
	пароль: 1234Emp
ПРИМИЧАНИЕ1: только администратор может добавлять/редактировать какие-либо объекты
ПРИМИЧАНИЕ2: '0' в таблице означает, что информация не указана
Главый экран для посетителей
Экран со всеми заявками для сотрудников служб(доступ получить могут только они)
Экран с полной информацией обо всём для администраторов
